# # 💼 Zawodowe - PHP & SQL Ultimate Pack

> "Zarządzanie strukturą firmy w czystym PHP – solidny fundament od G.T. i nowoczesny look Bootstrapa."

To repozytorium to mój poligon doświadczalny. Bazując na solidnych przykładach z lekcji u legendarnego Grzegorza T., postanowiłem dojechać temat PHP do perfekcji. Żadnych generatorów, żadnego lania wody – tylko czysta logika i walka o każdy rekord w bazie.

---

### 📜 Lore: Projekt Pracownia
Aplikacja **Pracownia** (a konkretnie moduł Pracownicy) to nie jest zwykły projekt. To system CRUD zbudowany od zera, żeby zrozumieć, jak naprawdę płyną dane między front-endem a bazą. To tutaj uczyłem się, że porządny SQL to podstawa, a `ID_SZEFA` w relacji to nie żarty.

*   **Fundament:** Techniczny konkret prosto z lekcji zawodowych.
*   **Misja:** Zrozumieć PHP 8.x pod maską, bez chodzenia na skróty przez frameworki.
*   **Vibe:** Lekkość, szybkość i pełna kontrola nad kodem.

### ⚡ Funkcje Systemu (Cheat Sheet)

| Funkcja | Status | Opis |
| :--- | :---: | :--- |
| **Full CRUD** | ✅ | Dodawanie, edycja i usuwanie (pracownicy, etaty, zespoły). |
| **Eagle Eye Search** | 🔍 | Błyskawiczne wyszukiwanie ludzi po imieniu i nazwisku. |
| **Relational Logic** | 🔗 | Twarde powiązania w bazie (ETAT, ID_SZEFA, ID_ZESP). |
| **Anti-Misclick** | 🛡️ | System potwierdzeń, żebyś przypadkiem nie zwolnił połowy firmy (szefa też, bo co to za pracownia bez szefa). |
| **Vanilla UI** | 🎨 | Nowoczesny layout z zakładkami. Czyste HTML/CSS i Bootstrap + Bootswatch. |

### ⚠️ Troubleshooting (Gdy baza powie "nie")
*   **Connection Error?** Sprawdź plik konfiguracyjny, pewnie host albo hasło leży.
*   **Puste tabele?** Nie zapomnij zaimportować struktury `.sql` przed startem.
*   **Style nie działają?** CTRL + F5 – przeglądarka czasem kocha stare cache bardziej niż Twój nowy kod.
