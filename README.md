# # 💼 Zawodowe - Projekty i Nauka

Repozytorium zawiera moje projekty zawodowe, zadania szkolne. Jest to zapis mojego rozwoju technicznego oraz przygotowań do egzaminu zawodowego INF.03.

---

### 🚀 Główny Projekt: System "Pracownia"
W folderze `/pracownia` znajduje się autorska aplikacja do zarządzania strukturą firmy:
*   **Zarządzanie:** Obsługa bazy danych pracowników, etatów oraz zespołów.
*   **Technologie:** Logika oparta na **PHP** oraz bazie **MySQL**.
*   **Interfejs:** Frontend zbudowany w **HTML** i **CSS** z wykorzystaniem frameworka **Bootstrap**.
*   **Stylizacja:** Personalizacja wyglądu za pomocą motywów **Bootswatch**[cite: 1].
